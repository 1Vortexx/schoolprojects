Part 2c — Dynamic List (Variable Length)
Step 1: Click Run Program — enter how many names you want
Step 2: Enter that many names when prompted
Result: All names printed vertically — run again with a different count!
Output will appear here after you run the program...
▸ View MakeCode TypeScript 
let names: string[] = []
let HowMany = game.askForNumber("How many names?")
for (let i = 0; i < HowMany; i++) {
    let Name = game.askForString("Enter a name:")
    names.push(Name)
}
for (let i = 0; i < names.length; i++) {
    game.splash(names[i])
}
