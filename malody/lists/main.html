02 Lists Assignment 1

Computer Science Principles — click a project to open it
project 1
Number List
List of 5 numbers. Press A to add 4 unique random numbers (0–50). Press B to print the list.
project 2
Word List
List of 3 words. A = add "before", B = add "after", Up = set center to "middle", Down = print vertically.
project 3
Random Animal Picker
List: dog, cat, fish, bird, hamster. Press A to pick a random animal by index using if statements.
project 4
Random Phrase Generator
6 adjectives + 4 nouns. Press A to generate a random adjective-noun phrase.
project 5
Treasure Collector Game
Move with arrow keys to collect 5 food sprites. Treasures list fills as you collect. At score 5, the list prints.
project 6
Mystery Gift Box
6 gifts in a box. Press A to reveal and remove a random gift. When empty, "The box is empty." ends the game.
